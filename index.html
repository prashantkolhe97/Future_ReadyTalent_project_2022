My Projects

1. To-do list

I have use JavaScript to build a web app that allows to make to-do lists for routine tasks. For this project, well-versed with HTML and CSS. JavaScript is the best choice for a to-do project since it allows users to design interactive coding lists where you can add, delete, and also group items.

2. Survey form

Building a survey form or questionnaire is easy. Even today, lots of companies use survey forms as a means of collecting relevant data about their target audience. In this project, I have to design a full-fledged survey form that includes relevant questions like name, age, email, address, contact number, and other questions, depending on the type of company or organization.

3. Countdown timer

Another simple project on our list is a countdown timer or clock. For this project, I just need to create a simple webpage that can update the time every second. With JavaScript as its foundation, I have make the page more appealing by including start, stop, and pause buttons on the page.

4. One-page layout

This project aims to recreate a pixel perfect design and make a one-page responsive layout. This is also a beginner-level project that allows freshers to test their newly acquired knowledge and skill level.

5. Product landing page

To develop a product landing page of a website. In this project, I have create columns and align the components of the landing page within the columns. I have to perform basic editing tasks like cropping and resizing images, using design templates to make the layout more appealing, and so on.
